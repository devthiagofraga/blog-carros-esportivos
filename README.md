# 🏎️ Esportivos BR - Blog de Carros de Performance

Este é um projeto de um blog dinâmico focado em carros esportivos de alta performance no Brasil. O sistema utiliza PHP e MySQL para gerenciar o conteúdo, apresentando um layout moderno e responsivo.

## 🚀 Funcionalidades
- **Grade Dinâmica:** Exibição de 9 posts recuperados diretamente do banco de dados MySQL.
- **Páginas Individuais:** Sistema de rotas dinâmicas para detalhes de cada veículo.
- **SEO Ready:** Estrutura de HTML5 semântico, metatags e páginas de Políticas de Privacidade.
- **Layout Responsivo:** Grid de 3 colunas que se adapta a diferentes resoluções.
- **Área de Publicidade:** Espaço reservado no rodapé para monetização.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** PHP 8.2
- **Banco de Dados:** MySQL (MariaDB)
- **Servidor Local:** XAMPP (Apache)
- **Estilização:** CSS3 (Flexbox e CSS Grid)
- **Integração:** PDO para conexões seguras.
